
☕ Coffee Shop Version two

This project is a simple and elegant coffee shop landing page I designed to reinforce my basic web development skills. I prioritized simplicity in design, aiming to give the user a peaceful coffee experience.

🛠 Technologies Used
This project was developed using only basic web technologies:

HTML5: Page structure and content hierarchy.

CSS3: Visual editing, color palette, and typography.

📌 Notes About the Project
Simplicity Focused: Created entirely with "vanilla" (pure) code, without the use of complex libraries or frameworks.

Development Process: This work is a learning phase project focusing on layout and styling practices.

Screen Compatibility: The project is not currently responsive (mobile-friendly). The best viewing experience is achieved on desktop browsers. Work on mobile compatibility is planned for future updates.
